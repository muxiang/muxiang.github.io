PowerLib.WPF

基于 WPF 的轻量级 Windows UI 库

Quick Start

1. 安装 NuGet 包

Install-Package PowerLib.WPF

2. 在 App.xaml 中添加资源

<Application.Resources>
    <ResourceDictionary>
        <ResourceDictionary.MergedDictionaries>
            <ResourceDictionary Source="pack://application:,,,/PowerLib.WPF;component/Themes/Generic.xaml" />
        </ResourceDictionary.MergedDictionaries>
    </ResourceDictionary>
</Application.Resources>

3. 在 XAML 中使用控件

<Window x:Class="MyApp.MainWindow"
        xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
        xmlns:pl="clr-namespace:PowerLib.WPF.Controls;assembly=PowerLib.WPF">
    <Grid>
        <pl:PrimaryButton Content="Hello PowerLib!" />
    </Grid>
</Window>
